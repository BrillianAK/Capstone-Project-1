![developer](https://img.shields.io/badge/Developed%20By%20%3A-Brillian-red) ![Python](https://img.shields.io/badge/Python%20Version-3.12-blue) ![Course](https://img.shields.io/badge/Purwadhika-DTI%20DS%20and%20ML%200206-maroon)

## Functions
### Admin
- Read Medical Record
- CRUD Medicine Stock
- Read and Delete User

### Doctor
- CRUD Appointment Schedule
- Update Patient Medical Record

### Patient
- Create Medical Record
- Read Patient Medical Record

## Data
### Column
- User
  - Name
  - Password
  - Role

- Medical Record
  - ID
  - Name
  - Appointment
  - Doctor
  - Diagnosa
  - Medicine

- Medicine
  - Name
  - Stock

- Doctor Schedule
  - Name
  - Schedule
  - Condition
